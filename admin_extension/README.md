# Existing web admin extension

The `legacy-*` files are the cache-safe versions of the existing admin entry
files. Copy them back as `admin.html`, `js/admin.js`, and `js/supabase.js`.
Also copy `admin-v2.js` and `admin-v2.css`.

`admin-v2.js` loads `admin-v2.css` automatically.

Pack cards receive their own publish/free/paid controls. Products contain
whole packs rather than individual assets.
